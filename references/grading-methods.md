# 批改方法论

> 使用时的入口：批改练习和交互练习时按需 Read 本文件。评分卡格式见 `exercise-template.md`。

## 苏格拉底提示阶梯（简化版 3 级，第 2 级跳过不用）

实际流程：指方向 → 答对过，还错 → 直接讲解。

| 用户答案 | 做法 |
|----------|------|
| ✅ 正确 | 一句话确认。"对了。下一题：..." |
| ❌ 错误，且是同一知识点第一次错 | **第1级-指方向**：不给答案，只给范围提示。如"看看时间——是过去还是现在？"→ 用户补答 → 对了就过，还错就升到第3级 |
| ❌ 错误，且补答后还是错 | **第3级-讲解**：直接给完整答案+原因，然后过 |

## 费曼检验

当用户连续2次错同类型题时：
```
你能用中文解释一下这个语法规则吗？
```
说不清楚的地方就是没真懂的地方。

某语法点正确率 >90% → 主动问"能一句话说清楚这个语法怎么用吗？"
说清楚→已掌握，说不清→巩固中。

## 概念对比卡模板

```markdown
⚡ **{A} vs {B}**
| | {A} | {B} |
|---|---|---|
| 职责 | {A做的事} | {B做的事} |
| 常见错误 | {易混点} |
| 课文例句 | {NCE课文中的例句} |
```

常用对比：
- 现在完成时 vs 一般过去时
- some vs any
- a/an vs the
- this/that vs these/those
- I do vs I am doing
- was doing vs did

## 误解诊断

不只判对错，分析错误类型：

| 错误类型 | 示例 | 修复方向 |
|---|---|---|
| 拼写错误 | "umbrela" | 抄写+音标对照 |
| 语法规则混淆 | "He don't like" | 对比卡+变式练习 |
| 词汇误用 | "I have 20 years" | 正确例句+场景替换 |
| 语序错误 | "I very much like" | 词序规则+句型操练 |
| OCR 误判 | "It4" 应为 "It's" | 提示用户原题可能有乱码 |

## 答案容错规则

批改时模糊匹配，宽容以下差异：
- 大小写不一致
- 多余/缺失空格
- 常见 OCR 误判映射（It4→It's、isnt→isn't、whats→what's）
- 中英文标点差异
- 同义答案（如 "I am" = "I'm"）
