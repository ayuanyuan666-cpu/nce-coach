# 学习报告

> 使用时的入口：用户选⑤或提到"学习报告"/"进度"/"报告"时 Read 本文件。从 `learner-state.json` 实时编译，直接在命令行输出，不出 MD 文件。

## 报告模板

```
## 📊 学习报告

| 项目 | 内容 |
|------|------|
| 当前进度 | Book X · Lesson Y / Z（X%） |
| 连续打卡 | N 天 🔥 |
| 最近练习 | L45 80% | L46 85% | ...（最近 5 条） |

### 语法掌握
已掌握（X项）：<列出>
巩固中（X项）：<列出>
薄弱（X项）：<列出>

### 待复习错题（N 条）
- 今天到期（M 条）：<列出截止日>
- 明天到期：...
- 后天+到期：...

### 薄弱词汇
<列表，无则显示"暂无">

### 🏁 里程碑
下一站：L48（1/3）还剩 2 课

### 💡 建议
<基于数据的一句 actionable 建议>
```

## 数据提取规则

- `grammar_mastery` 按 status 分三组列出（已掌握/巩固中/薄弱），分组名 + 语法点列表
- `review_queue` 按 `next_review` 日期分组：今天到期 / 明天到期 / 后续到期
- `exercise_history` 展示最近 5 条，格式：`L45 80%（2026-05-25）`，日期用完整年月日
- 里程碑按当前册总课次计算 1/3、1/2、2/3、终点四个节点，显示最近一个
- 建议生成逻辑：`ready_to_advance = true` → 建议新课；有薄弱语法点 → 建议巩固；全部正常 → 鼓励

## 输出后衔接

"以上就是你目前的学习数据。有想做的吗？" — 自然回到菜单。
