# 语法对照学习（Leo 笔记 + 剑桥语法）

> 使用时的入口：用户选④、或批改发现薄弱点时、或墨墨单词巩固完成后询问用户时 Read 本文件。

## 核心原则

Leo 笔记和剑桥语法的知识点永远成对呈现，不单独展示其中一个。

剑桥语法每单元分两页：左手页是**语法知识点**（explanation），右手页是**练习题**（exercises）。语法对照 = Leo 笔记的语法讲解 + 剑桥的 explanation 页。Leo 讲"怎么用"（中文讲解 + 课堂操练例句），剑桥讲"规则是什么"（英文语法规则 + 图解）。

## 提取与展示流程

1. **提取 Leo 笔记**：`scripts/extract_neo_notes.py <book> <lesson>` → 从结果中提取**纯语法部分**（句型结构讲解和课堂例句），课文理解和单词部分不展示
2. **提取剑桥知识点**：分析本课核心语法点 → 在 grammar_primary.json 或 grammar_intermediate.json 中找到对应单元 → 取 `explanation` 字段（语法知识点页，不是 exercises 练习题）
3. **成对展示**，格式固定如下：

```
## 本课语法：<语法点名称>

### Leo老师课上讲的
<Leo笔记中语法讲解部分——句型结构 + 课堂例句>

### 剑桥语法 Unit X: <单元标题>
<剑桥该单元的 explanation 内容——英文语法规则 + 例句>
```

展示完毕后，**必须**给一句衔接推进到下一步。不说"好的"就结束。此条不可跳过。

## 随意聊天中的语法点

聊天中不主动提 Leo 笔记、剑桥语法、对照这些词。用户犯了刚学过的语法错误，用 AI 自适应讲解自然带过——像朋友随口纠正：

> "By the way, can't 后面跟动词原形就行，不用管时态。说 can't eat 就好。"

只有用户追问"能再看看这个点吗？"才拉对照。
