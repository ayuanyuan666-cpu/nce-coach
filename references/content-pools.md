# 墨墨阅读材料：内容池

> 使用时的入口：墨墨单词巩固→用户选①阅读材料时 Read 本文件，用于选取形式和主题组合。

## 选取规则

每次生成从池中选一个形式+一个主题组合。

**轮换规则（强制）：**
1. **不连用同类**：与 `content_preferences.last_type` 不同形式、与上次主题不同类别
2. **20% 冷门探索**：20% 概率从整个池中随机选（跳过 liked 列表中的），防止内容同质化
3. **优先级排序**（80% 正常选择时）：
   - 第1优先：liked 评分 ≥ 2 的形式/主题
   - 第2优先：用户 interests/occupation 匹配的类别
   - 第3优先：从未用过的组合
   - 兜底：随机选一个非上次的组合
4. **记录机制**：每次生成后**立即**更新 `content_preferences.history`（保留最近 20 条），记录 `{date, format, theme, feedback}`

画像驱动原则：
- `interests` → 选匹配的类别
- `occupation` → 选职业相关主题
- `topics_mentioned` → 已聊过且正反馈的主题优先复用
- `english_goals` → 按目标匹配场景

生成前查看 `content_preferences`，优先用高评分类型和主题。生成后立即更新 history。
