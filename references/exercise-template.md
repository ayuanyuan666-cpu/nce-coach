# NCE 练习模板与批改格式

> 使用时的入口：出练习 MD 和批改时按需 Read 本文件。

## 练习 MD 作答格式

用户在题目下方以 `> ` 开头作答：
```md
**1.** This ___ my book. (is/are)
> is

**2.** 翻译：这是我的伞。
> This is my umbrella.
```

## 解析规则

1. 每个 `**N.**` 开头的行 = 一道题
2. 紧跟的 `> xxx` 行 = 用户答案
3. 没有 `> ` 的行 = 未作答
4. 多行连续 `> ` = 同一题的多行答案（如翻译题）

## 批改标注格式

批改直接写入 MD 文件。在用户 `> ` 答案行后追加：
```
> 用户答案
>> ✅ 对的
```
或
```
> 用户答案
>> ❌ 参考答案：... 讲解：...
```

单词造句部分答案不唯一，主要看句型结构对没对，直接在下一行标注语法点评。

## 成绩卡片（插入 MD 文件标题后、正文前）

```markdown
| 大题 | 得分 | 正确率 |
|------|------|--------|
| 单词造句 | 20/30 | 67% |
| 练习详解 | 9/9 | 100% |
| ... | ... | ... |

💪 最强项：can...but can't... 句型全对
⚠ 薄弱点：couldn't 时态区分、特殊疑问句倒装
```
