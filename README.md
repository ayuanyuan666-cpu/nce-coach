# Leo新概念英语私教

本学习方法来自于哔哩哔哩刘羽Leo老师的新概念英语教学体系。

## 这是什么

一个 AI 课后陪练，负责你学完新概念英语每课之后的练习批改、单词巩固和日常英语闲聊。它不教课（你有刘羽Leo老师的视频），只管课后陪你练。

## 能做什么

- **课后练习批改**：学完一课 → 自动聚合练习详解 + 剑桥语法 + Leo老师课堂练习 → 生成练习卷 → 你做完批改
- **单词造句+句型转换**：课后练习做四句型转换（肯定/否定/一般疑问/特殊疑问），墨墨巩固做三句型（肯定→否定→疑问），一句一句出
- **中英对照阅读**：用当日词汇生成段落交替的中英对照故事/对话，薄弱词加粗复现
- **随意英文聊天**：完成学习任务后进入双语聊天（英前中后），自然穿插语法纠正，记录兴趣画像

## 结合墨墨背单词

本 skill 可与 **墨墨背单词** 联动使用，形成单词学习闭环：

```
墨墨 App 背完单词 → Skill 拉取今日词汇
    → 精简展示（仅展开薄弱词，不列全表）
    → 阅读材料/句型转换巩固
    → 薄弱词自动跟踪沉淀（review_count、status 字段）
    → 顽固词生成个性化助记+例句写回墨墨
    → 第二天巩固时优先展示之前没掌握的词
```

需要在墨墨 App 内「设置 → 开放 API」申请 token，首次使用时提供给 Skill 即可。

## 依赖

- 新概念英语教材（课文 + 练习册）
- 剑桥英语语法 (English Grammar in Use)
- 墨墨背单词 App（可选，需自行申请 API token）
